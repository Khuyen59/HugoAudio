# Cách chạy project
**Tất cả các lệnh dưới đều phải mở bằng _Command Prompt_ trong project để chạy nhé**
## Nhớ cài node_module mới chạy được nhé

```bash
npm install
```
or 
```bash
yarn install
```

**B1: Chạy Metro**
```bash
npx react-native start
```
**B2: Chạy project**
```bash
npx react-native run-android
```
